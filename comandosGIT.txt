cd 'nome da pasta' -> para entrar em alguma pasta
cd.. ->para voltar um nivel
cd ->endereco da pasta completo
clear ->limpar tela do bash
git init ->cria um repositorio na pastas
git add 'nome do arquivo' -> adiciona o arquivo selecionado
git add *.'extensao' -> adiciona todos os arquivos da extensao em questao
git add . -> adiciona todos os arquivos do diretorio
git commit -m "Breve descricao do que foi feito" -> 
git commit -a -m "descricao" -> pula a etapa de git add
git diff -> mostra o que foi alterado
git diff --staged -> mostra o que foi alterado e que esta na staged
